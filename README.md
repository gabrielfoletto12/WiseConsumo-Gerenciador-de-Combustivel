# ⛽ WiseConsumo 

## - Descrição do Problema Real
Manter um veículo gera custos constantes que muitas vezes são subestimados. Seja para profissionais que dependem do carro para sua margem de lucro, ou para motoristas comuns que buscam saúde financeira, a falta de um controle rigoroso sobre o consumo de combustível e quilometragem impede a identificação de gastos excessivos, falhas mecânicas precoces e desperdício de recursos.

## - Proposta da Solução
O WiseConsumo  resolve essa "dor" ao oferecer uma ferramenta técnica e organizada para monitoramento automotivo. A aplicação permite que qualquer motorista registre seus abastecimentos de forma sistemática, transformando dados soltos em informações valiosas como médias de consumo real (km/l) e relatórios de gastos detalhados, promovendo uma gestão financeira inteligente e consciente.

## - Público-Alvo
- **Motoristas Profissionais**: Entregadores e motoristas de aplicativo (Uber, 99) que precisam controlar custos para garantir lucro.
- **Motoristas Comuns**: Proprietários de veículos de passeio que desejam um controle rigoroso sobre seu orçamento doméstico e economia de combustível.
- **Entusiastas de Manutenção**: Pessoas que utilizam a média de consumo como indicador preventivo para a saúde mecânica do veículo.

## - Funcionalidades Principais
- Registro de abastecimento com validação de quilometragem.
- Histórico completo ou filtrado por datas.
- Relatório de gastos totais, litros e preço médio ponderado.
- Cálculo automático de desempenho (km/l).

## - Tecnologias Utilizadas
- **Java 26**: Linguagem principal.
- **Maven**: Gestão de dependências e automação de build.
- **JUnit 5**: Testes automatizados.
- **Checkstyle**: Análise estática de código (Padrão Google).

## - Instruções de Instalação e Execução
1. Certifique-se de ter o JDK 26 e o Maven instalados.
2. Clone o repositório.
3. No terminal, execute: `mvn clean install`
4. Para rodar a aplicação: `mvn exec:java`

## - Instruções de Testes e Lint
- **Rodar Testes**: No terminal, use `mvn test`.
- **Rodar Lint (Estilo)**: No terminal, use `mvn checkstyle:check`.
